Magnetometer 3D Calibration Portal

System Instructions

To calibrate the compass:

  1. Click Start Calibration.
  2. Slowly rotate the Heltec Tracker in a figure-eight pattern in 3D space to map all orientations.
  3. Observe the raw data points filling the scatter plot on the right.
  4. Once you have mapped a clean sphere, click Stop Calibration (which the start button turns into), and review the computed parameters.
  5. Click Store Calibration to NVM to send and save values permanently.
